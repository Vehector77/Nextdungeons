# NextDungeons v2.0 - Complete Plugin

## 📋 Project Structure

```
src/main/
├── java/com/nextdungeons/
│   ├── plugin/             # Main plugin class
│   ├── api/                # Public API
│   ├── config/             # Configuration system
│   ├── dungeon/            # Dungeon system
│   ├── class_system/       # Player classes
│   ├── quest/              # Quest system
│   ├── party/              # Party management
│   ├── leaderboard/        # Rankings system
│   ├── items/              # Custom items
│   ├── rewards/            # Reward system
│   ├── hooks/              # Plugin integrations
│   ├── commands/           # Commands
│   ├── listeners/          # Event listeners
│   └── utils/              # Utilities
└── resources/              # Configuration files
```
